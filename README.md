# 🌍 WanderAI

AI-powered travel planning app with real-time chat, trip sharing, and mobile support.

## Features
- ✨ AI Trip Generator
- 💬 Real-time Group Chat
- 🗺️ Interactive Maps
- 📸 Trip Photos
- ⭐ Trip Reviews
- 📱 PWA Mobile Support

## Tech Stack
- React + Vite
- Supabase
- EmailJS
- Framer Motion

## Setup
1. Clone the repo
2. `npm install`
3. `cp .env.example .env`
4. Add your Supabase keys
5. `npm run dev`
